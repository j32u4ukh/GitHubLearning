# Issue and Projects

issue 類似於項目討論，可以由作者或是其他人發起，在該 issue 下進行討論與回覆。

有些人會將它應用在對專案進度的紀錄與追蹤等，類似於筆記的功能。

而 issue 條列式的呈現方式，在數量越變越多的情況下，會變得難以管理， Projects 的目的就是在對眾多 issue 進行分類與管理。
